## 웹과 서버의 통신 과정들에대한 이해
 - 클라이언트 : 사용자는 웹 브라우저를 통해 서버에 요청
 - 서버 : html 코드를 읽어 데이터베이스로부터 찾아서 사용자에게 전송
 - 데이터베이스 : 데이터들이 저장되어 있는 곳
 - PHP : 클라이언트, 서버, 데이터베이스들을 아울러 동적인 페이지를 관리
